This repository contains all the scripts to generate the results presented in this [manuscript](https://www.medrxiv.org/content/10.1101/2025.10.01.25337064v1) project is about modelling dengue vaccination in Singapore. We develope an age-stratified, serotype-stratified, dengue transmission model.
All the codes written in R (Version: R-4.3.1).
Following packages have been used for model run:
1. odin.dust 2. dust 3. mcstate
